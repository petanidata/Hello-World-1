Bahasa Program HTML 5

Pendahuluan

HTML 5 adalah sebuah pembaharuan untuk bahasa HTML seri sebelumnya. Banyak penambahan-penambahan code yang baru dan tentunya sangat menambah aksebilitas untuk penggunanya.
Dalam HTML 5 kita harus menggunakan <! DOCT html> untuk membuka bahasa ini dan ditempatkan paling awal baru setelahnya diisi oleh elemen-elemen lainnya.
Dan juga ditambahnkan pula elemen < meta charset="UTF-8" > yang berfungsi sebaga penanda type text yg kita input ke HTML, elemen ini ditempatkan setelah bagian head < head > yang akan kita bahas nanti.

Bagian-bagian pada HTML 5

pada HTML 5 terdapat 2 bagian yaitu:

  1. Bagian Head
    Bagian ini berisikan title yaitu tempat nama website kita. Title ini dapatkita input dengan query <title> dan jangan lupa untuk menambahkan tag penutup dengan cara menambahkan garis miring / lalu diikuti query nya. saya akan membuat contoh seperti ini:
    <head>
    <title>     Nama website kita </title>
    </head>
  2. Bagian Body
    Bagian ini berisikan head yaitu isi web kita, entah itu text, artikel, atau angka. Cara menggunakannya ialah dengan menambahkan query <Body> lalu kita isi text,angka,atau yang lainnya.
    Untuk mengakhirinya tambahkan saja garis miring / cara ini berlaku untuk mengakhiri semua query dalam HTML. Saya contohkan seperti ini:
    <Body>
    <p>     Isi website kita </p>
    </body>
Secara keseluruhan dapat kita susun seperti ini:
<!DOCT html>
<html>
<head>
<meta charset="UTF-8"> <title> nama website kita </title>
</head>
<body>
<p> isi website kita </p>
</body>
</html>

Tutorial

Nah kali ini kita masuk ke bagian Tutorial , hehehe langsung gigi 4 aja ya gann...

  1. Cara Menambahkan Gambar di HTML 5
    Di HTML 5 kita dapat menambhakan gambbar yang kita mau ke dalam website dengan menggunakan query
    <img src="gambar.jpg height="100px" width="100px" align="" border="1px" />
  2. Menambahkan File Audio
    Menambahkannya dengan cara memakai query
    <audio controls autoplay>
    <source src="musik.mp3" type="audio/mpeg">
    </audio>
    dan hasilnya akan keluar seperti ini
  3. Menambhkan Video
    Video dapat kita tambahkan/input dengan query <video controls autoplay>
    <source src="video.mp4" type="video/mp4">
    </video>
    dan hasilnya akan seperti ini
  4. Menambahkan Link
    Kita dapat menambahkan suatu link kedalam HTML dengan query
    <a href="www.link kita.com target="http//dsdsdad.com> Link </a>
    saya akan membuat contohnya seperti ini
    saya
  5. Membuat Tabel Sederhana
    Membuat tabel di HTML dapat menggunakan query
    <table border="2px">
    <tr>
    <td>    Kolom 1</td>
    <td>    Kolom 2</td>
    </tr>
    </table>
    Kira-kira hasilnya akan seperti ini
    Kolom 1 Kolom 2
  6. Membuat List
    Kita dapat membuat list sederhana dengan menggunakan elemen <ol> untul list yang berurut dan elemen <ul> untuk list yang tidak berurut.
    Disini saya akan membuat contoh list yang berurut:
    <ol>komposisi</ol>
    <li>    Minyak Nabati</li>
    <li>    Penguat Rasa</li>
    <li>    Gula</li>
    </ol>
    Sekarang kita akan membuat list yg tak berurut:
    <ul>Komposisi</ul>
    <li>    Minyak Nabati</li>
    <li>    Penguat Rasa</li>
    <li>    Gula</li>
    </ul>

    Bila kita masukan query ke dalam HTML maka akan menghasilkan seperti ini
    List berurut

    1. Minyak Nabati
    2. Penguat Rasa
    3. Gula

    list tidak terurut:
    • Minyak Nabati
    • Penguat Rasa
    • Gula
